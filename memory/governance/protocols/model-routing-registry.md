# Model Routing Registry

## Purpose

Consolidated policy truths for model routing, consultant escalation, channel authority, and disclosure discipline.

## Durable Rules

- Route by task category and risk, not by model hype.
- Keep cheaper tiers for low-risk work and reserve premium tiers for high-stakes reasoning.
- Require explicit disclosure and channel-appropriate communication format.
- Keep consultant requests auditable and linked to decision evidence.
