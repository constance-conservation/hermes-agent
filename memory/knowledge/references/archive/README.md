# Archive

This directory tracks superseded runtime-agent documents removed from active navigation.

## Archive Policy

- Keep active navigation focused on root anchors and active domain folders (`governance/`, `actors/`, `knowledge/`, `runtime/`).
- Preserve superseded material in git history and, when needed, in archived files here.
- Do not use archived files as primary policy inputs.

## Superseded Groups

- `* copy.md` variants
- old YAML backup snapshots (`*.bak*`)
- relocated template originals now represented under `runtime/tasks/templates/`
