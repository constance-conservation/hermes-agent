# Runtime

Domain for live execution context, procedures, and event history.

## Subdomains

- `state/` - current focus, runtime config, memory baseline
- `tasks/` - procedures, scripts, and templates
- `logs/` - governance logs, requests, and historical runtime traces

## Usage

- Load `state/current-focus.md` before executing tasks.
- Load relevant procedure from `tasks/procedures/`.
- Append outcomes to appropriate file in `logs/`.
