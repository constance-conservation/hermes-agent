# Delegation Packet Template

Use this packet for every delegated task.

## Required fields

- Task ID, parent task, delegating role, timestamp
- Assigned role and expected duration
- Model + reasoning level + fallback
- Allowed tools + forbidden tools + authority level
- Scope boundary (paths, network, budget)
- Objective, success tests, stop conditions
- Context capsule (minimal required context)
- Output format and escalation trigger

## Output requirement

Every delegate returns:

- concise result
- evidence links/paths
- blocker summary (if any)
- explicit next action
