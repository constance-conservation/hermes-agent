# Role Templates and Standards

Purpose: role-definition and lifecycle standards for governed org structure.

## Canonical hierarchy

Human Operator -> Chief -> Org/HR -> Directors -> Project Leads -> Supervisors -> Workers.

## Required role schema

- mission
- supervisor
- department
- project scope
- permissions
- success metrics
- lifecycle/deactivation criteria

## Memory locality

- workers: detailed task memory
- supervisors: workstream summaries
- project leads: milestone/blocker/evidence state
- directors: portfolio summaries
- chief: strategic active state

Archive inactive detail and keep active context lean.
