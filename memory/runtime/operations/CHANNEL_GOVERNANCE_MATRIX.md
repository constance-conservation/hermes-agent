# Channel Governance Matrix

Purpose: define channel hierarchy, posting permissions, and escalation etiquette.

## Priority order

1. WhatsApp DM: final high-priority outcomes only.
2. Telegram DM/group: strategic discussion.
3. Slack DM/channels/threads: operational execution substrate.

## Core principle

Resolve internally first; escalate to human only when policy/impact requires it.

## Posting authority

- Chief: all channels per governance.
- Security governor: security-critical escalations.
- Directors/leads/supervisors/workers: mainly Slack layer by role policy.

## Mandatory disclosure

All human-visible messages end with role signature:

`--<Role Name>`
