# Agent Creation and Lifecycle

## Purpose

Merged creation procedure for governed agents, profile-bound roles, approvals, and lifecycle tracking.

## Creation Tiers

- Tier 1: logical role in existing runtime
- Tier 2: isolated Hermes profile runtime
- Tier 3: full organizational spawn requiring architecture-level review

## Mandatory Sequence

1. Define role, mission, supervisor, scope, permissions, metrics, and lifecycle.
2. Register role before activation.
3. Instantiate minimal prompt shell and run first scoped task.
4. Track status in lifecycle register and schedule review checkpoints.

## Approval and Hygiene

- Prevent duplicate roles and vague scope.
- Require explicit reporting lines and measurable success criteria.
- Archive or suspend stale roles using deactivation criteria.
